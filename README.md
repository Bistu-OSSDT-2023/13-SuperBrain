# 13-SuperBrain
# 2048游戏简介：
《2048》是一款数字益智游戏，它的初始数字是基数2。通过不同方向的移动，相同数字的方块在靠拢、相撞时会相加，产生4，8，16等数字，玩家要想办法在16格范围中凑出「2048」这个数字方块。
